## Context / 为什么做

图像生成不应该只是输入一段提示词，而应该更接近自然创作：说出想法、即时调整、连续生成、逐步逼近。

## Process / 如何做

我负责把语音输入、意图理解、提示词生成、图像生成和结果反馈串成一个连续链路，让用户用更自然的方式驱动 AIGC。

## System / 系统结构

![AI 语音生图 Agent 视觉示意](/images/works/compound-practice/cover.svg)

- Voice Input：用语音表达创作意图
- Intent Parsing：识别风格、主体、场景和约束
- Prompt Builder：将自然语言转译成生成提示
- Generation Loop：连续生成、选择与微调

## Result / 结果

项目完成产品化验证，并在语音交互生图方向形成领先探索。它让我更确信，多模态交互会重塑 AI 创作产品。

## Reflection / 反思

真正好的 AI 交互，不是让用户学习机器语言，而是让机器理解人的表达方式。
